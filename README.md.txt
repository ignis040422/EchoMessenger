(C# 코딩) 에코 메신저
개요

C# 프로그래밍 학습

1종 소개: 사용자 키보드 입력을 받아서 처리하는 프로그램

사용한 플랫폼:

C#, .NET Windows Forms, Visual Studio, GitHub

사용한 컨트롤:

Label, TextBox, ListBox, Button

사용한 기술과 구현 기능:

Visual Studio를 이용한 UI 디자인
string 클래스를 이용한 사용자 입력 데이터 처리
DateTime 클래스를 이용한 현재 시간 정보 출력
수업 중에 배우고 사용했던 클래스 관련 설명
실습 중에 구현한 기능 설명

실행 화면 (과제1)

과제1 코드의 실행 스크린샷

"C:\Users\user\Pictures\Screenshots\1번 사진.png"

과제 내용

Label(표시), TextBox(입력), Button(전송), ListBox(대화창)를 적절히 배치한다.
전송 버튼 클릭 시 TextBox의 텍스트를 ListBox의 항목(Items)으로 추가한다.
추가 직후 TextBox의 내용을 비워(Clear) 다음 입력을 준비한다.

구현 내용 및 기능 설명

입력창에 메시지를 입력하고 전송 버튼을 누르면 메시지가 리스트 박스에 표시된다.
계속 반복하면 메시지가 리스트 박스에 한 줄씩 계속 추가된다.
추가 내용이 많아지면 리스트 박스에 스크롤바가 자동으로 생성되고 스크롤된다.
